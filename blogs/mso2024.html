Install Microsoft Office Profesional Plus LTSC 2024 Tanpa Langganan

Juni 15, 2023
By Admin TKJBlogs
Tutorial

Microsoft Office Professional Plus LTSC 2024 adalah versi terbaru dari suite produktivitas Microsoft yang dirilis untuk pengguna bisnis dan pendidikan. Berbeda dengan Microsoft 365 yang menggunakan model langganan, versi LTSC (Long-Term Servicing Channel) ini dapat dibeli dengan lisensi sekali bayar tanpa perlu berlangganan bulanan atau tahunan.

Persyaratan Sistem

Sebelum menginstal Microsoft Office Professional Plus LTSC 2024, pastikan komputer Anda memenuhi persyaratan sistem berikut:

  • Sistem Operasi: Windows 10 (versi 1909 atau lebih baru) atau Windows 11
  • Prosesor: 1.6 GHz atau lebih cepat, 2-core
  • RAM: 4 GB (64-bit)
  • Ruang Disk: 4 GB ruang disk kosong
  • Resolusi Layar: 1280 x 768
  • Grafis: DirectX 10 dengan WDDM 2.0 atau lebih tinggi

Langkah 1: Download Office Deployment Tool

Langkah pertama adalah mengunduh Office Deployment Tool (ODT) dari situs resmi Microsoft:

1. Kunjungi situs Microsoft dan unduh Office Deployment Tool:

2. Klik tombol Download dan simpan file officedeploymenttool.exe ke komputer Anda

3. Jalankan file tersebut dan setujui persyaratan lisensi

4. Pilih lokasi ekstraksi (misalnya: C:\ODT) dan klik "Extract"

Langkah 2: Buat File Konfigurasi XML

Selanjutnya, Anda perlu membuat file konfigurasi XML yang akan digunakan oleh ODT untuk mengunduh dan menginstal Office 2024:

1. Buka Notepad atau editor teks lainnya

2. Salin dan tempel kode XML berikut:

<Configuration> <Add OfficeClientEdition="64" Channel="PerpetualVL2024"> <Product ID="ProPlus2024Volume" PIDKEY="XXXXX-XXXXX-XXXXX-XXXXX-XXXXX"> <Language ID="id-ID" /> <ExcludeApp ID="Access" /> <ExcludeApp ID="Groove" /> <ExcludeApp ID="Lync" /> <ExcludeApp ID="OneDrive" /> <ExcludeApp ID="OneNote" /> <ExcludeApp ID="Teams" /> </Product> </Add> <Updates Enabled="TRUE" /> <Display Level="Full" AcceptEULA="TRUE" /> <Property Name="AUTOACTIVATE" Value="1" /> </Configuration> Tersalin!

3. Simpan file dengan nama "configuration.xml" di folder ODT yang sama (C:\ODT)

4. Catatan: Anda dapat menyesuaikan file konfigurasi ini sesuai kebutuhan:

  • Ubah "OfficeClientEdition" menjadi "32" jika Anda menggunakan sistem 32-bit
  • Ubah "Language ID" sesuai bahasa yang diinginkan
  • Hapus baris ExcludeApp jika Anda ingin menginstal aplikasi tersebut

Langkah 3: Download dan Install Office 2024

Sekarang Anda siap untuk mengunduh dan menginstal Microsoft Office 2024:

1. Buka Command Prompt sebagai Administrator

2. Navigasikan ke folder ODT dengan perintah:

cd C:\ODT Tersalin!

3. Jalankan perintah berikut untuk mengunduh file instalasi Office:

setup.exe /download configuration.xml Tersalin!

4. Setelah proses download selesai, jalankan perintah berikut untuk menginstal Office:

setup.exe /configure configuration.xml Tersalin!

5. Tunggu hingga proses instalasi selesai. Ini mungkin memakan waktu beberapa menit tergantung pada kecepatan komputer Anda.

Langkah 4: Aktivasi Office 2024

Setelah instalasi selesai, Anda perlu mengaktivasi Office 2024:

1. Buka Command Prompt sebagai Administrator

2. Navigasikan ke folder instalasi Office dengan perintah:

cd C:\Program Files\Microsoft Office\Office16 Tersalin!

3. Jalankan perintah berikut untuk mengaktivasi Office menggunakan KMS:

cscript ospp.vbs /inpkey:XXXXX-XXXXX-XXXXX-XXXXX-XXXXX Tersalin!

4. Ganti XXXXX-XXXXX-XXXXX-XXXXX-XXXXX dengan kunci produk yang valid

5. Jalankan perintah berikut untuk menetapkan server KMS:

cscript ospp.vbs /sethst:kms.example.com Tersalin!

6. Ganti kms.example.com dengan alamat server KMS yang valid

7. Aktifkan Office dengan perintah:

cscript ospp.vbs /act Tersalin!

Kesimpulan

Selamat! Anda telah berhasil menginstal dan mengaktivasi Microsoft Office Professional Plus LTSC 2024 tanpa langganan. Versi ini akan terus berfungsi tanpa perlu pembaruan langganan, meskipun tidak akan mendapatkan fitur baru seperti pada Microsoft 365.

Beberapa keuntungan menggunakan versi LTSC:

  • Pembayaran sekali tanpa biaya berlangganan
  • Tidak ada pembaruan fitur yang dapat mengganggu alur kerja
  • Dapat digunakan tanpa koneksi internet
  • Lebih stabil karena jarang diperbarui

Perlu diingat bahwa tutorial ini hanya untuk tujuan pendidikan. Pastikan Anda memiliki lisensi yang sah untuk menggunakan Microsoft Office Professional Plus LTSC 2024 di lingkungan produksi.
